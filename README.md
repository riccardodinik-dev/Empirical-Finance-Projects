# Empirical Finance – Python Projects

This repository contains a collection of Python notebooks developed during the *Empirical Finance* course of the MSc in Finance at Bocconi University.

The projects apply statistical, econometric and quantitative methods to financial datasets, with a particular focus on return modelling, probability, linear regression, factor models and dimensionality reduction.

The repository combines theoretical concepts with practical implementation in Python and illustrates how empirical methods can be applied to financial markets and investment analysis.

# Repository Structure

## Project 1 – Financial Prices and Return Analysis

**Dataset**

- IBM daily stock prices (Yahoo Finance)

**Objective**

Introduction to financial datasets and return computation.

Main topics include:

- Downloading market data using the Yahoo Finance API
- Data cleaning and preprocessing
- Simple and log returns
- Price dynamics
- Return visualization
- Descriptive statistics

------------------------------------------------------------------------------------------------------------------------------------------

## Project 2 – Log Random Walk Simulation

**Dataset**

- Simulated financial returns

**Objective**

Simulation and analysis of the Gaussian Log Random Walk model.

Topics covered include:

- Random number generation
- Gaussian innovations
- Log price dynamics
- Monte Carlo simulation
- Statistical properties of returns
- Long-run behaviour of asset prices

------------------------------------------------------------------------------------------------------------------------------------------

## Project 3 – Estimation of the Log Random Walk Model

**Dataset**

- Bank of America (BAC) daily prices

**Objective**

Estimation of return moments under the Log Random Walk assumption.

The notebook includes:

- Return estimation
- Sample mean and variance
- Annualization
- Sampling properties
- Estimator comparison
- Financial interpretation

------------------------------------------------------------------------------------------------------------------------------------------

## Project 4 – Distribution of Financial Returns

**Dataset**

- Bank of America (BAC) daily returns

**Objective**

Empirical analysis of the unconditional distribution of financial returns.

Topics include:

- Normality assessment
- Standardization
- Kurtosis
- Skewness
- QQ-plots
- Statistical tests
- Distributional properties of asset returns

------------------------------------------------------------------------------------------------------------------------------------------

## Project 5 – Multivariate Probability and Statistics

**Dataset**

- Simulated multivariate data

**Objective**

Introduction to multivariate statistics and linear algebra for finance.

The notebook covers:

- Matrix algebra
- Covariance matrices
- Correlation structures
- Vector operations
- Linear transformations
- Numerical implementation with NumPy

------------------------------------------------------------------------------------------------------------------------------------------

## Project 6 – Linear Regression for Corporate Finance

**Dataset**

- S&P 500 company fundamentals

**Objective**

Construction of predictive linear regression models for firms' payout ratios.

The notebook includes:

- Feature selection
- Data preprocessing
- Ordinary Least Squares (OLS)
- Model estimation
- Goodness-of-fit analysis
- Prediction performance

------------------------------------------------------------------------------------------------------------------------------------------

## Project 7 – Style Analysis and Factor Models

**Dataset**

- Multi-asset portfolio returns

**Objective**

Performance attribution through Sharpe's Style Analysis and benchmark factor models.

Main topics include:

- Portfolio style analysis
- Performance attribution
- Multi-factor regression
- Exposure estimation
- Investment style identification
- Risk factor interpretation

------------------------------------------------------------------------------------------------------------------------------------------

## Project 8 – Principal Component Analysis

**Dataset**

- Variance–covariance matrices and financial return data

**Objective**

Application of Principal Component Analysis (PCA) to financial datasets.

The notebook covers:

- Spectral decomposition
- Eigenvalues and eigenvectors
- Variance decomposition
- Dimensionality reduction
- Principal components
- Financial interpretation of common factors
